# MarketHub - Telegram E-commerce Bot

Bot Telegram per la vendita di prodotti digitali con pagamenti in criptovalute tramite NowPayments.

## Caratteristiche

- Catalogo prodotti con immagini
- Pagamenti in criptovalute (BTC, ETH, USDT, XMR, ecc.)
- Gestione ordini
- Hosting serverless su Vercel

## Configurazione

1. Imposta le seguenti variabili d'ambiente su Vercel:
   - `TELEGRAM_TOKEN`: Token del tuo bot Telegram
   - `NOWPAYMENTS_API_KEY`: API key di NowPayments
